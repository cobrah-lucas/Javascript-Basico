# 🚀 JavaScript Básico + Exercícios

![JavaScript](https://img.shields.io/badge/JavaScript-F7DF1E?style=for-the-badge&logo=javascript&logoColor=black)
![Status](https://img.shields.io/badge/Status-Em%20Desenvolvimento-brightgreen)

Repositório focado no aprendizado dos fundamentos do JavaScript. Contém exemplos práticos e exercícios sobre estruturas condicionais, laços de repetição, funções, manipulação de arrays e strings, além de pequenos projetos para fixação.

---

## 📂 Estrutura do Diretório

| Arquivo | Descrição |
|---------|------------|
| `estrutura-cond-js.html` | Exemplo de condicionais (`if`, `else if`, `else`) com base no nome do usuário. |
| `exercicio-array-for.html` | Exercício que percorre um array utilizando o laço `for` e exibe os dados. |
| `exercicio-nome.js` | Script simples que trabalha com entrada e manipulação de nomes. |
| `forJSrepeticao.html` | Demonstração do laço `for` aplicado a arrays. |
| `funcoes.js` | Declaração e uso de funções JavaScript (parâmetros, retorno, escopo). |
| `manip-arrays.js` | Operações básicas com arrays: adicionar, remover, filtrar, mapear. |
| `manip-string.js` | Métodos para manipular strings (`toUpperCase`, `slice`, `replace`, etc.). |
| `repeticao-while-do.html` | Exemplos das estruturas de repetição `while` e `do...while`. |
| `somajs.html` | Calculadora simples de soma entre dois números (entrada do usuário). |

---

## 🎯 Objetivo

Consolidar os conceitos essenciais da linguagem JavaScript antes de avançar para frameworks e bibliotecas. Cada arquivo aborda um tópico específico com código comentado e exemplos práticos.
